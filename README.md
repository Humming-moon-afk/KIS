# KIS - Clinical Information System 
- Project Goal: 
This project aims to mimic the core functionalities of a Hospital Information System (KIS).

# Backend (Java): 
- Focus on Object-Oriented Programming (OOP) and data validation to prevent clinical anomalies.

# Framework (Spring Boot): 
- Acts as the central nervous system, providing a REST API to handle JSON data exchange between frontend and database.

# Frontend (Web): 
- A user friendly interface built with HTML5 and CSS. 
- JavaScript is used to capture patient data and asynchronously communicate with the backend.

# Database (SQL): 
- Persistent storage for patient records and calculated scores using Spring Data JPA.
